1. Overview

A mobile-first marketplace connecting Customers with Traders (individual tradespeople or agencies/companies). The platform supports:

  • Service discovery, booking, and payment for customers
  • Job intake, crew management, earnings tracking, and payroll for traders
  • Agency/company workflows with group management, worker profiles, and automated payouts

The app renders role-specific experiences from a single codebase using a RoleBasedHome component that checks profile.role.

1.1 Service Classification

CategoryLabelDescription
Cat AQuick Fix / Fixed PriceStandard jobs with predefined pricing (e.g., tap repair, light switch replacement)
Cat BQuote / ProjectComplex jobs requiring a detailed quote from the trader (e.g., bathroom renovation)

4. Customer Flow

4.1 Home Page

  • Header: Location bar (tappable → address management) + notification bell
  • Search Bar: Full-text search → /search
  • Category Chips: Horizontal scroll of service categories
  • Hero Banner: Promotional banner with CTA
  • Service Bundles: Curated package deals (e.g., "Move-in Bundle")
  • Service Tabs: Toggle between Cat A and Cat B
  • Provider List: Top-rated traders with ratings, distance, availability
  • Saved Traders: Quick access to bookmarked traders

4.2 Booking Flow (5 Steps)

  1. Trader Selection — Choose specific trader or auto-assign
  2. Date & Time — Date grid (14 days) + time slots (09:00–17:00)
  3. Address — Select saved or enter new address
  4. Notes & Voice — Optional text notes + voice note recorder
  5. Payment & Review — Payment method, price with 21% BTW (VAT), confirm

Cancellation: Requires mandatory text-based reason, recorded in system.

5.3 Accept & Assign Flow (Cat A — Agency)

Bottom-sheet modal with 3 steps:

  2. Select Members: Review group members with checkboxes (all pre-selected)
  3. Confirm: Summary with avatars → accept + assign

5.4 Agency Quote Flow (Cat B — CollaborativeQuote) ★

This is the core quoting mechanism for agencies. When an agency clicks "Send Estimate" on a Cat B job, the job card is replaced inline with a multi-step flow:

Step 1: Choose Assignment

  • Assign to Group: Select from existing groups → proceeds to member review
  • Assign to Individuals: Multi-select search with chip tags

Step 2: Review Group Members

  • All members shown with checkboxes (pre-selected)
  • Group name + selected count in header badge
  • Deselect specific members as needed

Step 3: Confirm Assignment

  • Summary of assigned workers with "Will estimate" badge
  • Info banner: workers will receive notification to submit item estimates
  • "Assign & Request" button sends the estimation request

Step 4: Waiting

  • Animated worker avatars + pulse progress bar
  • "Collecting estimates…" with worker count

Step 5: Review & Send Quote

The review screen has these sections:

A. Average Estimate Banner — Shows calculated average across all submissions

B. Worker Item Estimates — Expandable per-worker cards showing itemized materials with quantities, prices, and totals

C. Final Materials (Averaged & Editable)

  • System auto-averages all worker estimates (same items → averaged qty/price)
  • Agency can edit descriptions, quantities, unit prices
  • Add/remove material line items

D. Labour Requirements (Type + Quantity Only)

  • Agency adds labour type (e.g., Plumber, Electrician, Helper)
  • Sets quantity only (number of workers needed)
  • No manual rate entry — base pay auto-resolved from group config:
    • £30/hr for skilled trades (default)
    • £25/hr for helpers/labourers
    • Configured via /trader/base-pay
  • Each line: role, count, auto base pay, line total

E. Totals Summary — Materials total + labour breakdown per type = Grand Total

F. Message — Optional text + voice note to customer

G. Submit — "Send Quote to Customer" (disabled if total ≤ 0)

5.5 Individual Trader Quote (Cat B)

Inline QuoteBreakdown component:

  1. Materials: add/remove items (description, qty, unit price)
  2. Labour: hours + hourly rate
  3. Message: optional text + voice note
  4. Total = materials + (hours × rate)

5.6 Worker-Side: Company Job Cards

Workers in an agency see company-assigned jobs with:

  • Company name badge + "Assigned" shield
  • Auto-accept timer (5 min countdown)
  • Budget hidden from workers
  • Masked customer location (area only)
  • Item estimation submission via WorkerQuoteRequest

5.7 Revenue & Earnings

  • Revenue summary with % change
  • Quick stats: Gross Income | Worker Pay | Platform Fees
  • Area chart with Week/Month/Year toggle
  • Unified transaction ledger with tags (Job Payment, Salary, Platform Fee)
